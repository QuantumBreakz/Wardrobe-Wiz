# WardrobeWiz

WardrobeWiz is a Vite + React front-end for a virtual try-on experience that blends AI styling, retrieval-augmented outfit generation, and sustainability insights. Everything runs purely in the browser right now so we can perfect the UX before wiring up the backend.

## Stack

- React 18 + TypeScript (Vite)
- Tailwind CSS + shadcn/ui components
- TanStack Query for future data hooks

## Getting started

```bash
pnpm install # or npm install / bun install
pnpm dev     # start the dev server on http://localhost:5173
```

## Scripts

- `dev` – start Vite development server
- `build` – production build
- `preview` – preview the production build
- `lint` – run ESLint (warnings remain for template files until cleaned up)

## Project status

- Authentication is prototype-only (admin / admin123).
- Dashboard routes showcase the final UX with mocked data.
- Additional pages (Product, Features, Pricing, FAQ, Company, Blog, Careers, Legal, Privacy, Terms, Security) provide content referenced in the footer.
